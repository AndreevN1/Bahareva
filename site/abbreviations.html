ПЕРЕЧЕНЬ СОКРАЩЕНИЙ И ОБОЗНАЧЕНИЙ

В настоящем дипломном проекте применяют следующие сокращения и обозначения:

АСУАвтоматизированная система управления
ВКСВидеоконференцсвязь
ГОГлавный офис (здание B1)
ДМЗДемилитаризованная зона
ИБПИсточник бесперебойного питания
ИСИнформационная система
КЖКабельный журнал
ЛВСЛокальная вычислительная сеть
МЭМежсетевой экран (Firewall)
ПКПерсональный компьютер
ПОПрограммное обеспечение
РМРабочее место
СКССтруктурированная кабельная система
СХДСистема хранения данных
ТКШТелекоммуникационный шкаф
ACLAccess Control List — список контроля доступа
ASAAdaptive Security Appliance — межсетевой экран Cisco (для моделирования в PT)
CCTVClosed-Circuit Television — система видеонаблюдения
DHCPDynamic Host Configuration Protocol — протокол динамической конфигурации хоста
DMZDemilitarized Zone — демилитаризованная зона
DNSDomain Name System — система доменных имён
FTPFile Transfer Protocol — протокол передачи файлов
HTTPHyperText Transfer Protocol — протокол передачи гипертекста
IPInternet Protocol — межсетевой протокол
L1/L2/L3Layer 1/2/3 — физический, канальный, сетевой уровни модели OSI
LANLocal Area Network — локальная вычислительная сеть
MGMTManagement — управление
PoEPower over Ethernet — питание по витой паре
PTCisco Packet Tracer — программа сетевого симулирования
RJ-45Registered Jack 45 — стандартный разъём для витой пары
STPSpanning Tree Protocol — протокол связующего дерева
SVISwitched Virtual Interface — виртуальный интерфейс коммутатора
UTPUnshielded Twisted Pair — неэкранированная витая пара
VLANVirtual Local Area Network — виртуальная локальная сеть
VPNVirtual Private Network — виртуальная частная сеть
WANWide Area Network — глобальная вычислительная сеть
