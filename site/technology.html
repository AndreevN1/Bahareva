4 ТЕХНОЛОГИИ ВЕБ

4.1 Подготовительный этап

Веб-сайт пояснительной записки разработан для централизованного представления всех разделов дипломного проекта в удобном и наглядном формате. Сайт предназначен для преподавателей, экзаменационной комиссии и студентов специальности 09.02.06.

На подготовительном этапе были определены: целевая аудитория, структура навигации (21 HTML-страница), требования к адаптивности, цветовая схема и шаблон макета. Макет был согласован с руководителем дипломного проекта.

4.2 Дизайн и цветовая схема

Цветовая схема разработана с использованием инструмента ColorScheme.ru в режиме «Моно» (оттенок 33°, тёплый коричневый). Схема обеспечивает профессиональный и строгий внешний вид.

4.3 Вёрстка сайта

Сайт свёрстан с использованием HTML5 и CSS3. Применены: Flexbox-компоновка для трёхколоночного макета (левое меню — контент — правое меню защиты), фиксированная шапка (position: fixed), адаптивные медиазапросы для экранов 1200px, 1024px и 768px. Все изображения имеют атрибут max-width: 100%, что предотвращает выход за границы контейнера. Таблицы обёрнуты в блок overflow-x: auto для горизонтальной прокрутки на узких экранах.

4.5 Публикация сайта

Сайт опубликован на платформе GitHub Pages — бесплатном хостинге статических сайтов от GitHub. GitHub Pages автоматически обслуживает HTML/CSS/JS-файлы из репозитория без необходимости настройки серверного ПО. Адрес публикации: https://[username].github.io/[repository]/site/. Для локального просмотра достаточно открыть файл index.html в любом браузере.
